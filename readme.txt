Integra nel tuo sito web il calendario di Optix per permettere ai tuoi clienti di prenotare gli appuntamenti in pochi
click!

== Description ==

Questo plugin ti permette di integrare il calendario del tuo gestionale per centro ottico Optix all'interno di qualsiasi
sito Wordpress. I tuoi clienti potranno prenotare un appuntamento nel tuo centro ottico direttamente dal tuo sito web in
pochi click. Il plugin fa uso del servizio esterno [Optix](https://optix.cloud/ "Optix"), puoi trovare maggiori
informazio sui termini e condizioni [qui](https://optix.cloud/terms/ "qui").

== Features ==

1. Supporta l'embed del widget all'interno delle pagine o articoli
2. Personalizza il widget permettendo di prenotare gli appuntamenti in tutti i tuoi negozio o solo in alcuni

== Shortcode ==

Puoi inserire il widget del calendario nelle tue pagine usando i shortcode. L'uso più semplice è
utilizzando `[optix-calendar]`.

== Frequently Asked Questions ==

= Per utilizzare il plugin devo avere un account Optix? =

Si, per poter usare il widget devi registrarti sul nostro sito https://optix.cloud. Farlo è gratuito!

= Cosa è l'API key? =

E' un codice molto importante che permette di far dialogare il plugin del calendario con il tuo account Optix. Il codice
è strettamente personale, pertanto non comunicarlo a nessuno!

= Dove trovo la mia API key? =

Accedi al tuo account Optix (https://optix.cloud), accedi alle Impostazioni>Integrazioni
e genera subito una chiave da utilizzare per integrare il sistema di prenotazione nel tuo sito.

= E' sicuro? =

Certo! Il nostro widget dialoga tramite protocollo sicuro e protetto con il tuo account Optix.
